Settings

GitHub Token

The GitHub Personal Access Token is used to save changes directly to the repository. Without it, the admin panel works in read-only / preview mode only.

Create a fine-grained PAT at github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens with Contents: Read & Write permission on the ethosobservatory/ethos-alumni repository.

The token is stored in your browser's localStorage. It is never sent anywhere except directly to api.github.com.

Admin Password

To change the admin password, update the ADMIN_HASH constant in admin.js with the SHA-256 hash of your new password. You can generate a SHA-256 hash online or with the command echo -n "your-password" | sha256sum.
